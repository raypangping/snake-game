# 贪吃蛇 HTML 小游戏 - 需求与技术栈

## 项目概述
创建一个基于 HTML5 Canvas 的经典贪吃蛇游戏，具有现代化的视觉效果和流畅的游戏体验。

---

## 功能需求

### 核心玩法
1. **蛇的移动**
   - 支持方向键（↑↓←→）和 WASD 键控制
   - 蛇向移动方向持续前进
   - 不能直接掉头（例如向右时不能直接向左）

2. **食物系统**
   - 随机位置生成食物
   - 吃食物后蛇身增长
   - 分数增加
   - 食物被吃后在新位置重新生成

3. **碰撞检测**
   - 撞墙游戏结束
   - 撞到自己身体游戏结束

4. **计分系统**
   - 实时显示当前分数
   - 显示历史最高分（localStorage 存储）

### 游戏控制
1. **开始/暂停**
   - 空格键开始/暂停游戏
   - 点击按钮开始/暂停

2. **重新开始**
   - 游戏结束后点击按钮重新开始

### 难度设置
1. **速度等级**
   - 简单：蛇移动速度慢
   - 中等：正常速度
   - 困难：快速移动

### 视觉效果
1. **蛇的样式**
   - 渐变色蛇身
   - 蛇头有眼睛区分方向
   - 移动动画流畅

2. **食物样式**
   - 发光/脉动效果
   - 随机颜色或特殊造型

3. **游戏界面**
   - 深色主题
   - 霓虹风格
   - 网格背景

### 音效（可选）
1. 吃食物音效
2. 游戏结束音效
3. 背景音乐（可选开关）

## 技术栈

### 核心技术
| 技术 | 用途 |
|------|------|
| HTML5 Canvas | 游戏渲染 |
| CSS3 | 样式和动画效果 |
| JavaScript (ES6+) | 游戏逻辑 |

### 关键技术点

1. **Canvas 渲染**
   - `requestAnimationFrame` 实现流畅动画
   - `ctx.fillRect()` 绘制蛇和食物
   - `ctx.arc()` 绘制圆形元素

2. **游戏循环**
   ```javascript
   function gameLoop() {
       update();  // 更新游戏状态
       draw();    // 绘制画面
       setTimeout(() => requestAnimationFrame(gameLoop), speed);
   }
   ```

3. **数据结构**
   ```javascript
   // 蛇身数组
   let snake = [
       { x: 10, y: 10 },  // 蛇头
       { x: 9, y: 10 },
       { x: 8, y: 10 }
   ];

   // 食物位置
   let food = { x: 15, y: 10 };

   // 移动方向
   let direction = { x: 1, y: 0 };
   ```

4. **碰撞检测**
   ```javascript
   // 撞墙检测
   if (head.x < 0 || head.x >= gridWidth ||
       head.y < 0 || head.y >= gridHeight) {
       gameOver();
   }

   // 撞自己检测
   for (let segment of snake) {
       if (head.x === segment.x && head.y === segment.y) {
           gameOver();
       }
   }
   ```

5. **本地存储**
   ```javascript
   // 保存最高分
   localStorage.setItem('snakeHighScore', highScore);
   // 读取最高分
   localStorage.getItem('snakeHighScore');
   ```
